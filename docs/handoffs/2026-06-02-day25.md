## Process lessons (Day 25 — reinforced hard)
- **Claude Code would not return literal git diffs again, AND repeatedly stalled in "ready to apply / Apply?" propose-mode instead of executing edits** — took several blunt "Yes, apply now" prompts to make it act. When it stalls: give a bare confirmation, then verify with grep + `npx tsc --noEmit; echo "EXIT: $?"` + a browser check.
- **Heredoc (cat > file << 'EOF') bypasses CC's edit-and-summarize drift entirely** — used it cleanly for new files. Prefer heredoc for NEW files; reuse/clone working files (e.g. the page.tsx guard) rather than hand-writing them.
- tsc EXIT 0 is the trustworthy code signal and caught both field-name errors — but code-verified is NOT feature-verified. Always browser-test (this morning's smoke test revealed adjustment had literally never been exercised).
- Currency convention is ৳ everywhere — match it; don't introduce Tk/BDT on one page.
