- **Claude Code would NOT return literal `git diff` output this session** despite repeated explicit asks — it summarized every time. Adjustment UI shipped verified-by-grep+tsc, not by literal diff. Demand the real diff first; fall back to grep+tsc if it won't come.

- Claude Code vs Supabase confused Arif. Rule of thumb: code (export/function/await/supabase client calls) -> Claude Code; raw SQL (SELECT/DELETE/ALTER/INSERT INTO) -> Supabase SQL Editor.
